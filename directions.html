Design directions for our website

Minimalist design

Our website has designed to ensure that our webpage has a small number of features so that the user does not get distracted.

Logo

The logo of our application has been featured prominently on the webpage. This will help users recognize our website and application easily. This way it can be distinguished from other applications.

Easy navigation

Our website is easy to navigate with a prominent navigation bar. The navigation bar does not contain self-links. We have designed it to highlight the page which the user is currently on. This will help the user remember where in the application they are.
